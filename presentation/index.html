Architecture Benefits

  • Clear separation of concerns
  • Horizontally scalable components
  • Stateless design for reliability
  • Modular tool-based approach
  • Multi-site MCP server support for extensibility

Module 1: MCP Server

Technology: Java 21 • Spring Boot 4.0.5 • Port 8080

Responsibilities

  • Implements recruitment tools via MCP protocol
  • Claude Haiku integration for cost-optimized AI
  • Zoho Recruit API integration with OAuth
  • Business logic for JD parsing and ranking

Key Components

Tools

MCP tool definitions and implementation

Services

AI and Zoho integration services

Module 2: MCP Client

Technology: Java 21 • Spring Boot 4.0.5 • Port 8081

Responsibilities

  • REST API gateway for frontend
  • MCP client communication with server(s)
  • Claude Haiku orchestration
  • Request/response transformation
  • Multi-site MCP server routing and load balancing

REST Endpoints

  • POST /api/job-descriptions/parse - Parse job description
  • POST /api/candidates/search - Search candidates
  • POST /api/candidates/rank - Rank candidates
  • POST /api/interviews/generate-questions - Generate questions

Module 3: React UI

Technology: React 19 • TypeScript 6.0 • Material-UI • Port 5173

Features

  • Job posting management and search
  • Candidate ranking visualization
  • Detailed fit analysis dashboard
  • Interview question management
  • Real-time candidate search and filtering

Technical Stack

State: Redux Toolkit
Build: Vite
Routing: React Router
API: Axios
