Wissen Recruit Ninja — Recruit Intelligence

Technical Innovation
Where We Push The Envelope
🔗
First-Mover MCP Adoption
We use Spring AI 2.0-M4's Streamable HTTP transport — one of the first enterprise MCP implementations in Java. Haiku orchestrates tools it has never seen before at runtime. No hardcoded pipelines.
// application.yaml — MCP Server
spring.ai.mcp.server
  protocol: streamable
  transport: http
🧠
Chain-of-Thought Explainability
Every scoring prompt forces the model to reason before it scores. The explanation block precedes the JSON output, eliminating hallucination and ensuring transparent, auditable hiring decisions.
// Structured output schema
{ rankPosition: 1,
  matchPercentage: 92,
  matchReasoning: "...",
  missingSkills: ["K8s"] }
⚡
Parallel Batch Execution
For speed-critical scenarios, candidates are sharded into batches of 10, processed simultaneously across 4 threads using a managed ExecutorService. 31 candidates ranked in 8–10 seconds flat.
// Parallel ranking — Java 21
executorService.submit(
  () → analyzeBatch(batch, jd)
); // × 4 threads
🛡️
Graceful Degradation by Design
When Anthropic goes down or rate-limits hit, our backend falls back to a validated regex-based parser and keyword-criterion builder. The product never experiences full downtime — enterprise SLA compliant.
// AIEnhancedRankingService.java
catch(AiException e) {
  return regexFallback(candidates);
}
