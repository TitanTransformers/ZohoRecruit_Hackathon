# Recommended Stack (Best for Claude Code)

Use:

* Frontend: Next.js + Tailwind + shadcn/ui
* Backend: FastAPI OR Next.js API routes
* DB/Cache: Redis (optional)
* Vector Search: pgvector / Pinecone / local embeddings
* LLM: Claude API (best for prompting quality)
* Embeddings: OpenAI text-embedding-3-large OR Voyage AI
* Auth: Internal only (simple)
* Deployment: Vercel + Railway / Render
* Zoho Recruit API: OAuth 2.0 REST API

Claude Code works extremely well with:

* TypeScript
* Next.js
* Python FastAPI
* well-structured monorepo
