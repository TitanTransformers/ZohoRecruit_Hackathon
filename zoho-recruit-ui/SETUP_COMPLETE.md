## 🔄 Data Flow

```
User Input
    ↓
React Component (DocumentUploadPage)
    ↓
Redux Actions (setText, setPdfFile)
    ↓
Redux Store (documentSlice)
    ↓
Component Re-render
    ↓
Fetch API POST Request
    ↓
Spring Boot API
    ↓
Process Request
    ↓
JSON Response
    ↓
Redux Actions (setSuccess/setError)
    ↓
Component Re-render
    ↓
User Sees Result
```
